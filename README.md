# OrekGames Claude Hooks

Welcome to the **OrekGames Claude Hooks** repository! 

This project is intended to host all public OrekGames [Claude Code](https://docs.anthropic.com/en/docs/agents-and-tools/claude-code/overview) hooks that we felt are fun, useful, or interesting enough to share with the community. 

As we develop new hooks to improve our agentic workflows or simply to add a bit of character to our daily coding, we will publish them here.

---

## 🪝 Available Hooks

| Hook | Description |
|---|---|
| **[claude-code-sounds](./claude-code-sounds)** | Play Warcraft 3 (or custom) sound bites when Claude Code triggers events (e.g., task complete, errors, tool usage). Features cross-platform Python scripts for random or context-aware sound playback. |

## 🚀 Usage

Each hook in this repository is intended to be completely self-contained within its own directory. 

To use a specific hook:
1. Navigate into the directory of the hook you are interested in.
2. Read the dedicated `README.md` located in that folder for specific setup, installation, and uninstallation instructions.

## 🤝 Contributing

We welcome contributions! If you have built an awesome Claude Code hook, or if you want to improve one of our existing hooks, feel free to submit a Pull Request or open an Issue.
